# Boxhead

Introduction: 
Boxhead is a Java application for a 2D Version of Call of Duty Zombies.
It is family-guy themed, using Stewie as the main player and Meg and Lois 
as zombies. The main player's goal is to survive increasingly difficult
rounds of zombies, using weapons that they pick up from killing more and more zombies. 
Uses location-recognition algorithm for zombies to follow player's location.
Includes 10+ variations to levels by increasing the frequency of the zombies and 
improving the timing and speed of their decision making.
